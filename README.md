# Щоденник Здоров'я (Health Journal)

Сучасний мобільний додаток на Flutter для відстеження артеріального тиску та рівня цукру в крові. Створений для зручного щоденного моніторингу та формування професійних звітів.

## Основні можливості

- **Контроль показників**: запис верхнього/нижнього тиску, пульсу та рівня цукру (з точністю до 0.1).
- **Аналітичні графіки**: візуалізація динаміки стану здоров'я на головному екрані.
- **PDF Звітність**: формування детального звіту за останні 30 днів з підтримкою української мови для передачі лікарю.
- **Розумні нагадування**: 
  - Автоматичні сповіщення о 8:00 та 20:00 (налаштовуються).
  - Функція повтору кожні 10 хвилин протягом години.
  - Інтелектуальна система: нагадування приходять тільки якщо дані за відповідний період ще не були внесені.
- **Персоналізація**: можливість змінювати час нагадувань або повністю їх вимикати у бічному меню.

## Як встановити на iPhone

Оскільки додаток знаходиться у режимі розробки, для встановлення на iPhone скористайтеся цією інструкцією:

### 1. Підготовка пристрою
1. Відкрийте **Налаштування** (Settings) на вашому iPhone.
2. Перейдіть до розділу **Приватність та безпека** (Privacy & Security).
3. Прокрутіть до самого низу та виберіть **Режим розробника** (Developer Mode).
4. Увімкніть перемикач і підтвердіть перезавантаження телефону.
5. Після перезапуску на запит телефону виберіть **"Увімкнути"**.

### 2. Встановлення (через комп'ютер розробника)
1. Підключіть iPhone до комп'ютера за допомогою кабелю.
2. Якщо з'явиться запит "Довіряти цьому комп'ютеру?" — натисніть **"Довіряти"**.
3. На комп'ютері запустіть команду:
   ```bash
   flutter run --release
   ```
4. Дочекайтеся завершення збірки та встановлення.

### 3. Підтвердження сертифіката
Після встановлення іконка з'явиться на робочому столі, але при спробі запуску з'явиться повідомлення про "Ненадійного розробника".
1. Перейдіть у **Налаштування** -> **Загальні** -> **VPN і керування пристроями**.
2. Виберіть ваш Apple ID у розділі "Програми розробника".
3. Натисніть **"Довіряти..."** та підтвердіть сертифікат.

**Тепер додаток готовий до використання!**
